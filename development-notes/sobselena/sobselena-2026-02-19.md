Первоначальная настройка проекта RSS Tandem

Были добавлены и настроены следующие скрипты:

- `npm run dev` — запуск Vite dev-сервера с HMR

- `npm run build` — сборка TypeScript и production-бандла в dist

- `npm run preview` — предпросмотр production-билда

- `npm run lint` — запуск ESLint для проверки кода

- `npm run format:fix` — автоматическое форматирование кода с помощью Prettier

- `npm run prepare` — настройка Husky (выполняется после установки зависимостей)

- `npm run style` — слежение и компиляция Sass

- `npm run test` — запуск тестов через Vitest

- `npm run test:ui` — запуск тестов с визуальным интерфейсом Vitest UI

- `npm run test:coverage` — запуск тестов с генерацией отчёта по покрытию

- `npm run validate-branch` — проверка корректности названия Git-ветки
